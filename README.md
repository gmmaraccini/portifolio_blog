Projeto 2026 - Projeto 1 Blog

. Sistema de Blog (ou CMS Simples)
O que faz: Permite que um "admin" crie, edite e delete posts. Visitantes podem ler os posts e deixar comentários (que precisam de aprovação).
Habilidades que demonstra: Operações CRUD (Create, Read, Update, Delete), interação com banco de dados (MySQL/PostgreSQL), sistema de login básico (autenticação), e sanitização de dados (para evitar XSS em comentários).
